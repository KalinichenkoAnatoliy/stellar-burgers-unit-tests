# 🍔 Stellar Burgers: Юнит-тестирование

Этот репозиторий содержит решение задания по юнит-тестированию в рамках дипломного проекта. 
Основная цель — покрытие тестами класса `Burger` сервиса заказа бургеров Stellar Burgers.

## 🎯 Задача
1. Написать юнит-тесты для класса `Burger`.
2. Использовать библиотеки `pytest` и `pytest-cov`.
3. Применить **моки (mocks)** для изоляции зависимостей.
4. Использовать **параметризацию** для проверки различных сценариев.
5. Обеспечить **100% покрытие кода (code coverage)**.

## 🛠 Стек технологий
- **Язык:** Python 3.x
- **Фреймворк для тестов:** pytest
- **Инструмент покрытия:** pytest-cov
- **Ветка разработки:** `develop1`
